# Consent Demo (Expo)

Consent Demo is a mobile-first UI prototype for consent and preference management built with Expo.

## Screenshot
![Screenshot](docs/screenshot.png)

Replace `docs/screenshot.png` with a real screenshot or GIF.

## Quickstart
### Prerequisites
- Node.js 20+
- npm
- Expo Go app (optional for mobile testing)

### Run locally
```bash
npm install
npx expo start
```

Use the Expo CLI to launch on Android, iOS, or web.

## Tests
```bash
npm test
```
